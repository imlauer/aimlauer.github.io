Como rutear todo el trafico de Internet a traves de Tor

25.01.2023
Índice

Instala Tor

sudo apt install -y tor
### Configura una DNS local

Agrega esto al final en el archivo /etc/tor/torrc

VirtualAddrNetwork 10.192.0.0/10
AutomapHostsOnResolve 1
TransPort 9040
DNSPort 53

De esta forma vas a configurar un servidor DNS en el puerto 53 y un proxy transparente en 127.0.0.1:9040

Lo siguiente a hacer es agregar esto al final de /etc/resolvconf/resolv.conf.d/head

nameserver 127.0.0.1

Luego necesitamos actualizar nuestro /etc/resolv.conf ejecutando este comando:

sudo resolvconf -u

De esta forma, vas a evitar cualquier fuga de DNS de su sistema.

Configuración del Firewall

Ahora tenes que configurar tu firewall de la forma en que cada conexion pase por el proxy transparente excepto el usuario Tor usando este script.

Para obtener el UID del usuario Tor ejecuten

id debian-tor

En mi caso el usuario es debian-tor, pueden fijarse el UID del usuario en /etc/passwd de la siguiente forma:

cat /etc/passwd | grep tor

Esto lo van a cambiar en la variable TOR_UID del siguiente script

#!/bin/sh

# destinations you don't want routed through Tor
NON_TOR="192.168.1.0/24 192.168.0.0/24"

# the UID Tor runs as
TOR_UID="109"

# Tor's TransPort
TRANS_PORT="9040"

iptables -F
iptables -t nat -F

iptables -t nat -A OUTPUT -m owner --uid-owner $TOR_UID -j RETURN
iptables -t nat -A OUTPUT -p udp --dport 53 -j REDIRECT --to-ports 53
for NET in $NON_TOR 127.0.0.0/9 127.128.0.0/10; do
 iptables -t nat -A OUTPUT -d $NET -j RETURN
done
iptables -t nat -A OUTPUT -p tcp --syn -j REDIRECT --to-ports $TRANS_PORT

iptables -A OUTPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
for NET in $NON_TOR 127.0.0.0/8; do
 iptables -A OUTPUT -d $NET -j ACCEPT
done
iptables -A OUTPUT -m owner --uid-owner $TOR_UID -j ACCEPT
iptables -A OUTPUT -j REJECT

Aviso: Este script lo van a tener que ejecutar en el arranque.

Un mejor método alternativo

Sigan leyendo la wiki oficial, hay un tipo de ataque en contra de este metodo y la solucion es usar IsolatingProxy.
